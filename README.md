# Restaurant Website

A fully responsive restaurant website featuring a dynamic menu, category filters, and an interactive shopping cart.

## Features

- Multi-page structure (Home, About, Menu, Contact)
- JavaScript-based category filters (Pizza, Burger, Sandwich)
- Interactive shopping cart with quantity controls
- Real-time total price updates
- Fully responsive across all devices

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6+)
